### Method 2: Manual Deploy
1. Install Vercel CLI:
   ```bash
   npm install -g vercel
   ```

2. Clone this repository:
   ```bash
   git clone https://github.com/syedabersabil/web-games-collection.git
   cd web-games-collection
   ```

3. Deploy:
   ```bash
   vercel
   ```

4. Follow the prompts and your games will be live!

## ğŸ® How to Play

### Snake Game
- Use **Arrow Keys** to control the snake
- Eat food to grow longer
- Don't hit the walls or yourself!
- Press **Space** to restart

### Flappy Bird
- Click or press **Space** to flap
- Navigate through pipes
- Score points by passing pipes

### Tic-Tac-Toe
- Click cells to place X or O
- Get three in a row to win
- Toggle between AI and 2-player mode

### 3D Cube Runner
- Use **Arrow Keys** to move left/right
- Press **Space** to jump
- Avoid obstacles and survive!

### Space Shooter 3D
- Move with **Arrow Keys**
- **Mouse** to aim
- **Click** to shoot asteroids
- Don't let asteroids hit you!
